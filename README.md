# Front-end Cionça Pindados

## Para instalar as dependências:
```
yarn install
```

### Para densenvolvimento:
```
yarn serve
```

### Para produção
```
yarn build
```

### Para rodar o lint:
```
yarn lint
```

Observação: Para a ideia de negócio funcionar é preciso estar rodando a [API](https://github.com/OncaPintada/api)
